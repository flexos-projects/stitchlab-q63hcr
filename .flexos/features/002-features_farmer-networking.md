# Core Features: Farmer Networking

## User Profiles
- **Create & Edit Profile**: Farmers can create detailed profiles including:
    - Name, Location (farm address/region)
    - Farm Type (e.g., dairy, crop, livestock, organic)
    - Interests (e.g., farming techniques, equipment, social activities, hobbies)
    - Photos (profile picture, farm photos)
    - Bio/About Me section
    - Looking For (e.g., romantic partner, business collaborator, mentorship, friendship)
- **View Profiles**: Users can view other farmers' profiles to learn about them.

## Matching & Discovery
- **Swipe Mechanism**: Users can 'swipe' left (pass) or right (like) on other farmer profiles.
- **Location-Based Matching**: Prioritize showing profiles of farmers within a certain radius.
- **Interest-Based Matching**: Suggest profiles based on shared farming interests or 'Looking For' criteria.
- **Mutual Match**: Only allow messaging when both users have 'liked' each other.

## Messaging
- **Chat Interface**: A private messaging system for mutually matched farmers.
- **Text & Emojis**: Basic text and emoji support within chat.

## Notifications
- **New Match Notification**: Alert users when they have a new mutual match.
- **New Message Notification**: Alert users when they receive a new message.

## Settings & Privacy
- **Profile Visibility**: Users can control who sees their profile (e.g., active, hidden).
- **Distance Preferences**: Users can set their preferred matching distance.
- **Account Management**: Basic account settings (password reset, delete account).
